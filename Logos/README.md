## Logo Usage Rules

See `02_Brand_Identity/logo_guidelines.md` for full rules. Key points:
- Never stretch or distort the logo
- Maintain minimum clear space around the mark
- Do not place on busy backgrounds without a background blocker
- Use white version on dark backgrounds
- Use black version on light backgrounds
