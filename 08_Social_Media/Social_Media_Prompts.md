## THE SOCIAL MEDIA BRAND CONTEXT BLOCK
Paste this into any social content prompt:

```
MVQUEEN SOCIAL MEDIA BRAND CONTEXT:
Brand: MVQUEEN — modern feminine luxury lifestyle brand.
Sister brand: Miss.Princess (softer, playful, younger — never mixed with MVQUEEN).
Core belief: Luxury was always yours. Every woman deserves to feel elevated.
Voice: Warm, confident, specific, poetic. Lead with feeling before product.
Short sentences hit harder. Em dashes (—) create rhythm.
The best MVQUEEN caption makes her want to screenshot it or send it to a friend.
FORBIDDEN: "empowering," "boss babe," "slay," "you've got this," passive permission language,
panic urgency, generic adjectives without sensory backup.
```

## INSTAGRAM CAPTION PROMPTS

### IG-01 — Product Feature Caption
```
[BRAND CONTEXT BLOCK]
Pillar: [Pillar 01–07 from Content_Pillars.md]
Product: [NAME] | Category: [skincare/fragrance/fashion/beauty/haircare]
Key feeling: [what she feels — specific] | Key detail: [one sensory/functional detail]
CTA: [yes "Shop via link in bio" / no]

Write 3 variations:
A — Atmospheric: scene or feeling opens, product arrives mid-caption
B — Identity: who she is or becoming, product is the vehicle
C — Sensory: specific texture, scent, or physical detail opens

Each: first line standalone hook / 80–160 words / ends with landing line / no hashtags in body
```

### IG-02 — Brand Essay Caption
```
[BRAND CONTEXT BLOCK]
Topic: [e.g., "getting dressed for yourself" / "soft life philosophy"]
Angle: [what MVQUEEN specifically believes]
Tone: conviction-led, not self-help. Real thought, not motivational poster.

Write 1 essay caption:
- Opens with one bold belief statement
- Develops over 3–4 short paragraphs
- Lands on a sentence she'll want to save
- 150–250 words | No CTA — this earns the follow, not the click
```

### IG-03 — Collection Launch Caption
```
[BRAND CONTEXT BLOCK]
Collection: [NAME] | Story: [2-sentence emotional world] | Hero product: [NAME]
Launch type: [drop/seasonal/collab] | Available: [now/early access/limited]

Write: 1. Launch day caption / 2. Hero product caption / 3. Final week soft availability
```

## TIKTOK / REELS SCRIPT PROMPTS

### TT-01 — Tutorial / How-To
```
[BRAND CONTEXT BLOCK]
Topic: [e.g., "5-step morning skincare ritual"] | Products: [list]
Length: [15/30/45/60s] | Hook type: [private thought/scene/bold claim/contrast]

Script format:
[0–3s]: Hook — on-screen text + spoken opening (stops the scroll)
[3–Xs]: Tutorial — step by step, conversational, product naturally featured
[end]: "Link in bio." Never urgency.
Voice: real woman showing something she genuinely loves. Not an ad.
```

### TT-02 — Atmosphere / Slow Edit
```
[BRAND CONTEXT BLOCK]
Pillar: [Feminine Atmosphere / Romanticized Living / Sensory Luxury]
Scene: [e.g., "morning skincare, warm light, slow motion"]
Mood: [e.g., "unhurried, quiet"] | Sound: [e.g., "soft ambient, warm trending sound"]
Length: 15–30s

Write: 1. On-screen text (max 1 line, 8 words) / 2. Caption (30 words, atmospheric) / 3. Sound direction
Note: No voiceover needed.
```

### TT-03 — Product Feature / Discovery
```
[BRAND CONTEXT BLOCK]
Product: [NAME] | The one thing: [most compelling detail] | Length: [15/30s]

Script:
[0–3s]: Hook — not the brand name, not "introducing"
[3–Xs]: Product — what it is, what it does, one sensory detail
[end]: Natural CTA
Voice: "I've been asked about this fragrance four times this week."
```

### TT-04 — Miss.Princess TikTok
```
MISS.PRINCESS CONTEXT: Younger, playful, trend-aware. More energy. Shorter sentences.
Emojis: 1–2 max. Can reference soft girl / princess era aesthetics.

Product: [NAME] | Aesthetic: [soft girl/princess era/cute beauty] | Length: [15/30s]

Script: trend-aware hook → enthusiastic product commentary → "Grab yours — link in bio 💕"
```
