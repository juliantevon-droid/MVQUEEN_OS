# 🚫 MVQUEEN — Forbidden Words & Phrases

---

## Why This Document Exists

Words carry energy. The wrong word in MVQUEEN copy — no matter how good the surrounding content is — breaks the atmosphere.

This document exists to protect the brand's voice from the language patterns that make luxury brands sound cheap, generic, or out of alignment with their values.

Every person and AI system producing MVQUEEN content must review this list before publishing.

---

## Tier 1 — Never Use (Hard Prohibition)

These words and phrases are incompatible with MVQUEEN brand identity. They signal cheap, aggressive, or generic marketing and must never appear in customer-facing content.

| Word / Phrase | Why Forbidden |
|--------------|--------------|
| BOGO / Buy One Get One | Discount retailer energy |
| Flash sale | Desperate urgency, contradicts luxury |
| Limited time offer! | Manufactured panic, aggressive |
| Don't miss out! | Fear-based, pressure marketing |
| Act fast! | Pushy, unbecoming |
| Hurry! | Same as above |
| Slay | Overused, performative |
| Boss babe | Dated, contradicts brand voice |
| Girlboss | Same as above |
| Hustle | Contradicts softness philosophy |
| Grind | Same as above |
| Literally (in brand copy) | Too casual, filler word |
| Obsessed (as headline) | Overused, meaningless |
| You NEED this | Pressure-based |
| Game changer | Marketing cliché, hollow |
| Life changing | Overused, unsubstantiated |
| Amazing | Generic, no specificity |
| Incredible | Same as above |
| Stunning | Overused in beauty/fashion |
| Iconic (without context) | Hollow without earned meaning |
| Synergy | Corporate filler |
| Leverage (in brand copy) | Jargon |
| Disrupt / Disrupting | Tech startup language, off-brand |
| Empower (overused) | Has lost meaning through overuse |
| Fierce | Contradicts softness and calm |
| Baddie | Not aligned with brand identity |
| Bestie (in brand copy) | Too casual for luxury positioning |
| Slaying the game | Multiple violations |
| It's giving | Trend-dependent, fleeting |
| No cap | Same |
| Lowkey / Highkey | Same |

## Tier 2 — Use With Extreme Caution

These words are not forbidden but are so overused in beauty/fashion marketing that they require a specific, earned context to carry weight. If used, they must be supported by real substance.

| Word / Phrase | Caution |
|--------------|---------|
| Luxury | Overused — only use when describing a specific feeling or experience, not as a generic label |
| Elevated | Common — pair with specific sensory detail |
| Exclusive | Needs to mean something real, not just signaling |
| Premium | Generic — replace with specific quality descriptor |
| Empowered / Empowering | Has lost specificity — describe the actual feeling instead |
| Authentic | Overused — show authenticity, don't label it |
| Intentional | Used well in doctrine, but can become filler in copy |
| Emotional (as prefix) | Brand philosophy uses this well, but copy overuse dilutes it |
| Feminine | Use precisely — don't use as a vague adjective |
| Self-care | Overused category term — describe the specific ritual instead |
| Glow | Beauty cliché — use only with specific sensory context |
| Radiant | Same as glow |
| Transform / Transformation | Powerful but requires a real before/after to earn it |
| Ritual | Used correctly in brand voice but can become a filler word |
| Journey | Overused, vague |
| Community | Use only when describing a real community interaction |

---

## Tier 3 — Never Use in Headlines or Hero Copy

These are acceptable in body copy with context but must never lead a headline, subject line, or primary hook — they are too weak to carry the opening moment.

- *Good* (as in "good products")
- *Nice*
- *Great*
- *Wonderful*
- *Perfect*
- *Beautiful* (alone, without supporting context)
- *Special*
- *Unique* (without specificity)
- *Quality* (without specificity)
- *Affordable luxury* (contradicts brand positioning)

## Punctuation & Formatting Prohibitions

| Pattern | Why Forbidden |
|---------|--------------|
| Multiple exclamation marks!!! | Desperate energy |
| ALL CAPS WORDS in body copy | Aggressive, shouting |
| 🛍️🔥💯 (transactional emojis) | Contradicts luxury positioning |
| "Click here" | Lazy, no brand voice |
| Ellipsis overuse... | Amateur writing pattern |
| "I mean..." in brand copy | Too casual for any format |
| Sentence fragments used as filler | Only acceptable as intentional stylistic choice |

## Words We Replace

| Instead of... | Use... |
|--------------|--------|
| Amazing | Extraordinary / Remarkable / [Specific descriptor] |
| Luxury (generic) | [Describe the specific feeling of luxury] |
| Empowering | Restoring / Honoring / Elevating |
| Self-care | Ritual / Practice / Intention |
| Beautiful (alone) | [Describe what makes it beautiful specifically] |
| Game changer | [Describe the specific change it creates] |
| Obsessed | In love with / Can't put down / Returns to every day |
| Stunning | [Describe what creates the impact] |
| Good vibes | [Describe the specific atmosphere] |
| Fierce | Confident / Grounded / Present |
| Boss | Intentional / Decisive / In her power |

## AI Content Review Checklist

Before any AI-generated content is published, scan for:

```
□ Any Tier 1 words present? → Remove and replace
□ Any Tier 2 words used without specific context? → Revise
□ Any Tier 3 words in headlines? → Replace with stronger language
□ Multiple exclamation marks? → Remove
□ ALL CAPS in body copy? → Remove
□ Generic CTAs (Shop Now, Buy Now)? → Replace with CTA library options
□ Overuse of "emotionally" as a prefix? → Rewrite naturally
□ Any language that sounds like another brand? → Rewrite as MVQUEEN
```
