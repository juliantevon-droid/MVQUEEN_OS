# 👑 MVQUEEN — Product Naming System
### Naming Conventions, Formulas & Examples

---

## What This File Is

Product names in MVQUEEN are not labels. They are the first piece of copy a customer reads — and they either pull her in or they don't.

This file governs how every product in the MVQUEEN ecosystem is named: the formulas, the rules, the emotional logic, the category-specific conventions, and real examples across every product type.

Every name must pass one test: *Does it sound like it belongs in the MVQUEEN world — or could it belong to any other brand?*

---

## THE NAMING PHILOSOPHY

MVQUEEN product names operate in one of three registers:

**Register 01 — Evocative**
The name creates a feeling or image before the product is described.
Best for: hero products, fragrance, beauty, fashion statement pieces.
> Examples: *Velvet Hour, Soft Reign, Ivory Ritual, Still Morning*

**Register 02 — Descriptive-Poetic**
The name communicates what the product is — but in elevated, brand-specific language.
Best for: skincare, haircare, everyday essentials, accessible entry products.
> Examples: *The Glow Serum, Soft Reset Mask, Deep Nourish Treatment*

**Register 03 — Identity-Led**
The name speaks to who the customer is or who she becomes when she uses it.
Best for: collections, bundles, fashion categories, lifestyle products.
> Examples: *The Becoming Edit, Queen's Ritual Set, The Soft Life Collection*

## PART 1 — NAMING FORMULAS BY CATEGORY

### FRAGRANCE NAMING

Fragrance names are the most evocative in the catalog. They should feel like a place, a moment, a memory, or an emotion — not a description of notes.

**Formula A — The Moment**
`[Time / Atmosphere] + [Emotional Qualifier]`
> Velvet Hour / Still Evening / Gold Dusk / Soft Midnight / Warm Silence

**Formula B — The Feeling**
`[Emotional State] + [Optional Noun]`
> Already Arrived / Quiet Confidence / Soft Power / The Knowing

**Formula C — The Place**
`[Sensory Place] + [Descriptor]`
> Ivory Room / The Garden, Warm / After the Rain, Soft / Still Water

**Formula D — The Single Evocative Word**
> Murmur / Satin / Aura / Reign / Vesper / Reverie / Ember

**Naming Rules — Fragrance:**
- Never name a fragrance after its notes (*Jasmine & Amber* is an ingredient list, not a name)
- Never use generic luxury words alone (*Luxe, Elite, Premium, Gold* without context)
- The name should make her feel something before she smells it
- Miss.Princess fragrances can be more playful: *Petal Dream, Pink Sugar Hour, Soft Spell*

---

### SKINCARE NAMING

Skincare names balance emotional resonance with functional clarity. The customer needs to understand what it does — but through language that makes her want to use it.

**Formula A — The Ritual Name**
`The + [Function] + [Product Type]`
> The Glow Serum / The Soft Reset / The Morning Ritual / The Deep Restore

**Formula B — The Emotional Function**
`[What it does for how she feels] + [Product Type]`
> Confidence Serum / Calm Recovery Mask / Soft Start Cleanser / Still Water Toner

**Formula C — The Sensory Name**
`[Sensory Descriptor] + [Product Type]`
> Velvet Skin Cream / Silk Cleanse / Warm Glow Oil / Dewy Veil Serum

**Formula D — The Minimalist**
Single word or short phrase that becomes iconic with repetition:
> Glow / The Serum / Soft / Restored / Still / Luminous

**Naming Rules — Skincare:**
- Include what it does (glow, restore, calm, hydrate) but through elevated language
- Avoid clinical terminology in the name (*Retinol Complex* is an ingredient, not a name)
- A number system can work for a sequential routine: *Step One / Step Two* or *Morning / Evening*
- Miss.Princess skincare can use softer names: *Dewy Dreams, Petal Skin, Soft Cloud Cream*

---

### FASHION NAMING

Fashion names in MVQUEEN operate at the piece level and the collection level. Pieces get names that describe the emotional role they play — not the garment construction.

**Formula A — The Role Name**
`The + [Who She Becomes In It]`
> The Arrival Dress / The Sunday Blazer / The Quiet Statement Coat / The Already Decided Set

**Formula B — The Feeling Name**
`The + [How It Feels]`
> The Soft Exit / The Unhurried Dress / The Still Sunday / The Weighted Silk

**Formula C — The Descriptor + Garment**
`[Elevated Adjective] + [Garment Type]`
> Ivory Slip / Dusty Rose Midi / Champagne Satin Set / Warm Cashmere Coat

**Collection Names:**
Collections get identity-level names — not seasonal descriptors:
> The Becoming Collection / Soft Reign / Ivory Hour / The Ritual Edit / Still Life

**Naming Rules — Fashion:**
- Individual pieces can carry the color as part of the name when color is central to identity
- Avoid describing the garment construction in the name (*Ribbed Knit Top* → *The Soft Rib*)
- Collections should feel like worlds, not product assortments

---

### HAIRCARE NAMING

Hair naming follows skincare logic — function through elevated language.

**Formula A — The Treatment Name**
`[What It Restores] + [Product Type]`
> Strength Ritual Mask / Soft Restore Treatment / The Deep Nourish / Shine Recovery Oil

**Formula B — The Sensory Name**
`[How It Feels] + [Product Type]`
> Silk Wash / Velvet Condition / Featherlight Spray / Warm Repair Mask

**Formula C — The Minimalist**
> The Mask / Restored / Silken / The Treatment

---

### LIFESTYLE / HOME NAMING

Lifestyle products use the most atmospheric naming — they are experience objects, not functional items.

**Formula:**
`[The Atmosphere They Create]`
> Still Evening Candle / Warm Room Reed Diffuser / The Morning Ritual Set / Soft Space Body Oil

## PART 2 — COLLECTION NAMING SYSTEM

Collections are the highest-level naming in the catalog. They organize individual products into a world with a single emotional identity.

### Collection Name Criteria
Every collection name must:
1. Create an atmosphere — not just describe products
2. Be emotionally distinct from every other collection
3. Work as a hashtag, a campaign headline, and a section header
4. Not expire with a trend

### The Collection Name Formulas

**Formula A — The Feeling**
`[Emotional State / Atmosphere]`
> Soft Reign / Still Life / The Arrival / The Becoming / Ivory Hour / Velvet Season

**Formula B — The Editorial**
`[Poetic phrase that creates a world]`
> Women Who Already Know / The Quiet Luxury Edit / For the Woman You're Becoming

**Formula C — The Time**
`[Time of day or season as emotional experience]`
> Golden Hour / The Still Morning / After Midnight / First Light / Warm Season

**Formula D — The Identity**
`[Who she is when she shops this collection]`
> The Softness Edit / The Confident Woman Collection / The Ritual Set / Elevated Essentials

### Miss.Princess Collection Names
> Soft Girl Summer / The Pink Edit / Princess Era / Dreamy Season / Petal Collection / Soft Spell

## PART 3 — NAMING RULES

### Rule 01 — Every Name Is Unique
No two products in the MVQUEEN catalog share a name or a name that sounds like another. Uniqueness is non-negotiable.

### Rule 02 — "The" Is a Power Tool
Starting a product name with "The" signals intention and specificity:
> *The Glow Serum* feels more considered than *Glow Serum*
Use it for hero products and statement pieces. Not everything can be "The" — it loses power if overused.

### Rule 03 — No Generic Luxury Words Alone
*Luxe, Elite, Premium, Gold, Diamond, Prestige* — these words are banned as standalone product names. They communicate nothing specific about MVQUEEN.

### Rule 04 — Avoid Trend-Dependent Names
Names that reference current TikTok language, meme culture, or temporary trends will date the product. MVQUEEN names are built for longevity.

### Rule 05 — Pronounceability Test
Every name must be easy to say aloud — in a review, in a recommendation, in a caption. If it requires explanation to pronounce, reconsider.
> ❌ "Lumière Sérum Intense" (over-engineered, pretentious)
> ✅ "The Glow Serum" (direct, confident, MVQUEEN)

### Rule 06 — Miss.Princess Names Are Distinct
Miss.Princess names are softer, more playful, and may use more color-forward or trend-adjacent language. They are never shared with MVQUEEN products.

## PART 4 — SKU NAMING CONVENTION

For internal inventory and Shopify management, every product uses the following SKU structure:

```
[BRAND]-[CATEGORY]-[PRODUCT CODE]-[VARIANT]

BRAND:    MVQ = MVQUEEN | MXP = Miss.Princess
CATEGORY: SKC = Skincare | FRG = Fragrance | FSH = Fashion
          HRC = Haircare | BEA = Beauty | LFS = Lifestyle
PRODUCT:  3-digit sequential number (001, 002, 003...)
VARIANT:  Size or colorway code (30ML, 50ML, IVR, RSE, BLK)
```

**Examples:**
- `MVQ-SKC-001-30ML` — MVQUEEN Skincare Product 001, 30ml
- `MVQ-FRG-003-50ML` — MVQUEEN Fragrance Product 003, 50ml
- `MVQ-FSH-012-IVR` — MVQUEEN Fashion Product 012, Ivory colorway
- `MXP-BEA-004-RSE` — Miss.Princess Beauty Product 004, Rose colorway

SKU codes are internal only. They never appear in customer-facing copy.

## PART 5 — NAMING CHECKLIST

Before finalizing any product name, run it through this checklist:

- [ ] Does it sound like it belongs specifically in MVQUEEN?
- [ ] Does it create a feeling before describing a function?
- [ ] Is it unique — not shared with any existing product?
- [ ] Is it easy to say aloud and recommend?
- [ ] Does it avoid generic luxury words used alone?
- [ ] Is it built for longevity — not trend-dependent?
- [ ] If it's Miss.Princess, does it carry her distinct energy?
- [ ] Does it work as a caption, a hashtag, and a search term?
