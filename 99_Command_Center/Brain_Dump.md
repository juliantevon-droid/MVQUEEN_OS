# 🧠 MVQUEEN — Brain Dump

---

## Purpose

Rapid capture for ideas, insights, observations, and decisions that need to be processed into the OS — but haven't been yet.

Everything that lands here gets processed within 72 hours.
Brain dump → sorted → actioned or archived.

---

## How to Use

Dump anything here:
- Ideas that came to you mid-shower
- Observations about the brand that feel important
- Competitor moves worth noting
- Customer insights from comments or DMs
- Product ideas
- Content ideas
- System improvements
- Problems that need solving
- Decisions that need to be made

**Format:** Date + brief description + any context
**Rule:** Don't over-structure here. This is the capture layer. Organize later.

---

## Processing Protocol

Every Sunday: Review brain dump.
For each item:
```
→ Actionable now? → Move to priority_tasks.md
→ Needs research? → Move to 26_RnD_Lab/
→ System improvement? → Move to relevant system folder
→ Content idea? → Move to Content Calendar
→ Product idea? → Move to 04_Products/Product_Framework.md
→ Not relevant? → Delete
```
