## Topic Filter Framework

Before creating content on any trending topic, run it through this filter:

**Step 1 — Relevance check**
Does this topic relate to: beauty / skincare / haircare / fashion / feminine lifestyle / self-care / intentional living?
If no → skip.

**Step 2 — Brand alignment check**
Does this topic align with: softness, luxury, femininity, restoration, intentionality?
If no → skip.

**Step 3 — Differentiation check**
Can MVQUEEN bring a unique angle to this topic that no other brand is bringing?
If no → wait for a better angle.

**Step 4 — Audience check**
Is this topic relevant to at least one MVQUEEN persona?
If no → skip.

**Step 5 — Timing check**
Is this topic still trending, or has it peaked?
Use TikTok and Google Trends data to confirm timing.
