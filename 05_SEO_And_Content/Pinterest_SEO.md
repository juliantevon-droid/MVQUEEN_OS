## Pinterest SEO Fundamentals

Pinterest is a visual search engine. SEO principles apply directly.

### Where Pinterest Indexes
| Location | Importance | Optimization |
|----------|-----------|-------------|
| Profile name | High | Brand name + keyword |
| Profile bio | High | Keyword-rich description |
| Board names | High | Searchable, keyword-led |
| Board descriptions | Medium | 2-3 sentences, keyword-rich |
| Pin title | Highest | Primary keyword first |
| Pin description | High | 100-200 words, natural keywords |
| Image alt text | Medium | Descriptive, keyword-aware |
| Hashtags | Low-Medium | 3-5 relevant tags |

## Pin SEO Optimization

### Pin Title Formula
`[Primary Keyword] — [Brand Value or Emotional Promise]`

Examples:
- *Luxury Face Serum — The Morning Ritual That Changes Everything*
- *Feminine Skincare Routine — Soft Skin, Intentional Living*
- *Soft Life Aesthetic — How to Build a Luxury Morning Ritual*

### Pin Description Formula
```
[Keyword-rich opening sentence — states what this is clearly]
[2-3 sentences of value/description — natural, not stuffed]
[Brand connection sentence — 1 line]
[Soft CTA]
[3-5 hashtags at end]
```

**Example:**
*This luxury face serum is designed for women who want more from their skincare ritual.*
*With a lightweight texture that absorbs instantly, it restores softness without heaviness.*
*Part of the MVQUEEN morning ritual — products built for real women's real mornings.*
*Shop the full ritual at the link.*
`#luxuryskincare #skincarerituals #femininebeauty #softlife #morningroutine`

## Content Strategy for Pinterest

### Volume
Pinterest rewards consistent, high-volume pinning.
Target: 5-10 new pins per day using scheduler (Tailwind or native).

### Content Types
| Type | % of Content | Description |
|------|------------|-------------|
| Product pins | 30% | Direct product images with shop links |
| Lifestyle pins | 40% | Aesthetic, mood, soft life content |
| Educational pins | 20% | Tips, guides, ingredient education |
| Blog pins | 10% | Drive blog traffic to site |

### Evergreen vs Trending
- 70% evergreen (skincare routines, morning rituals, feminine lifestyle)
- 30% seasonal (linked to campaign calendar)
