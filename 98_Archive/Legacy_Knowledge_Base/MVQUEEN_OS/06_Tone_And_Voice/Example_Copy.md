## The Standard These Examples Set

Every piece of copy in this library passes five tests:

1. **The Identity Test** — Could any other brand have written this? If yes — it was rewritten.
2. **The Centering Test** — Is she the center? Always.
3. **The Feeling Test** — Does it create a feeling before it delivers information? Always.
4. **The Voice Test** — Was it read aloud? Did it sound like MVQUEEN? Yes.
5. **The Forbidden Words Test** — Checked against `Forbidden_Words.md`. Clean.

New copy does not graduate to published until it passes all five.
