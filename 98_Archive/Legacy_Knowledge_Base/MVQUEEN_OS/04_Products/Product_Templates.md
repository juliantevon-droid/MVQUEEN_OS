# 📝 MVQUEEN — Product Copy Templates

---

## Purpose

Every product in MVQUEEN requires the same set of copy assets.
These templates ensure every listing is complete, consistent, and built to the enterprise standard before it goes live.

Do not publish a product without every section completed.
An incomplete product page is an incomplete brand experience.

---

## Template: FAQ Block (for product page FAQ section)

```
Q: Will this work for [skin/hair type]?
A: [Specific, warm, honest answer. Not "yes for all types" — be specific.]

Q: How do I use this?
A: [Ritual framing — make it sound like a ritual, not instructions]

Q: How long will one [unit] last?
A: [Honest estimate based on recommended usage]

Q: Is this suitable for sensitive skin/color-treated hair?
A: [Honest answer — if yes, why. If no, say so clearly.]

Q: What makes this different?
A: [One sentence — the MVQUEEN differentiator for this specific product]
```

## Pre-Publish Quality Checklist

Before any product goes live on Shopify:

- [ ] All 10 template sections completed
- [ ] Short and long descriptions checked against `Forbidden_Words.md`
- [ ] Both descriptions read aloud — pass the voice test
- [ ] SEO title under 60 characters
- [ ] Meta description under 160 characters
- [ ] All product images have alt text
- [ ] Minimum 8 Shopify tags applied
- [ ] Cross-sell and upsell products assigned
- [ ] SKU created in `SKU_Systems.md` and logged in `Product_Catalog.md`
- [ ] Product price confirmed against `Pricing_Strategy.md`
- [ ] Collection assigned in Shopify
