## 01 — Purpose of This File

This is the enforcement layer of the brand identity system. Where `color_system.md` and `typography_system.md` define *what to use*, this file defines *how to use it* — and what is never permitted.

Every person who creates anything under the MvQueen or miss.queen name — designer, copywriter, AI agent, VA, social media manager — is bound by this document.

---

## 02 — Brand Hierarchy

```
00_Doctrine/         ← Constitutional layer (why we exist, what we stand for)
     ↓
02_Brand_Identity/   ← Identity layer (what we look like and sound like)
     ↓
All execution        ← Everything made, posted, sold, or sent
```

**Rule:** Doctrine overrides identity. Identity overrides execution. When in conflict, defer up.

---

## 03 — The MvQueen Brand Pillars

Every piece of brand output must reflect at least one of these three pillars:

| Pillar | What It Means in Practice |
|---|---|
| **Quiet Confidence** | The brand never shouts. Copy doesn't beg. Design doesn't overcrowd. The product speaks. |
| **Warm Luxury** | Premium without being cold. Elevated without being alienating. She feels *chosen*, not sold to. |
| **Feminine Precision** | Everything is intentional. Color, spacing, word choice — nothing is accidental or generic. |

If a piece of content cannot be traced to one of these three pillars, it should not be published.

## 04 — Visual Brand Rules

### 4.1 — Logo
| Rule | Detail |
|---|---|
| Always use approved logo files | Never recreate the wordmark from fonts |
| Clear space | Minimum 1x logo height on all sides |
| Minimum size (digital) | 120px wide |
| Minimum size (print) | 1.5 inches wide |
| Approved backgrounds | White, Vault Black, Soft Cream |
| Prohibited backgrounds | Blush, Gold, any photo without contrast guarantee |
| Color variations | Full black, reversed white, gold version (packaging only) |
| Never | Stretch, rotate, recolor, add shadow, outline, or use low-res versions |

### 4.2 — Color (Summary Rules — Full Detail in color_system.md)
| Rule | |
|---|---|
| Primary palette only | Do not introduce colors outside the approved system |
| Gold is an accent | Never use as primary background for large areas |
| Flat color only | No gradients, no color overlays on imagery |
| Decorative colors (Blush, Nude) | Never for text or interactive elements |
| miss.queen palette | Never cross-applied to MvQueen assets |

### 4.3 — Typography (Summary Rules — Full Detail in typography_system.md)
| Rule | |
|---|---|
| Google Fonts system only | Cormorant Garamond, DM Sans, Italiana |
| Cormorant max weight | SemiBold (600) — never Bold or Black |
| H4 labels | Always uppercase, always tracked |
| Body copy | Left-aligned for paragraphs — never centered |
| No system font substitutions | Arial and Times are off-brand |

### 4.4 — Imagery
| Rule | Detail |
|---|---|
| Tone | Warm, natural light. Never harsh flash or clinical backgrounds |
| Models | Inclusive, confident, composed. Not editorial-cold. |
| Backgrounds | White, cream, warm neutral, or lifestyle context |
| Editing | Warm white balance. Slight desaturation on backgrounds. Product color must be true. |
| Text-on-image | Only with sufficient contrast overlay. Never on a busy image region. |
| Stock photography | Not approved for primary brand use — sourced or produced assets only |

### 4.5 — Spacing & Layout
| Rule | Detail |
|---|---|
| Breathing room | Generous white space is a brand signal — do not crowd |
| Grid | 12-column grid on desktop; 4-column on mobile |
| Section padding | 80px desktop / 48px mobile (minimum) |
| No visual clutter | One primary focal point per layout section |
| Borders | Use `--mvq-stone` at 1px or `--mvq-cream` dividers — never thick borders |

## 05 — Verbal Brand Rules

### 5.1 — Voice Pillars (Summary — Full Detail in 06_Tone_And_Voice/)
| Pillar | In Practice |
|---|---|
| Direct | Say the thing. No filler words, no hedging. |
| Warm | Never clinical, never cold. She's talking to a friend who happens to be an expert. |
| Precise | Every word earns its place. Cut what doesn't serve. |

### 5.2 — Copy Rules
| Rule | |
|---|---|
| No exclamation marks in brand copy | Exception: customer-facing celebrations (order confirmations, wins) |
| No "just" or "simply" | These words diminish — the brand does not minimize itself |
| No passive voice in headlines | Headlines are active and direct |
| Product names | Always title case, never all-caps |
| Brand names | MvQueen (not MVQUEEN, not Mv Queen, not mvqueen) |
| | miss.queen (always lowercase, with period) |
| No slang or trend-dependent language | Brand copy must age well |
| No urgency manipulation | "Last chance!" "Only 3 left!" — not in brand voice unless factually true |

### 5.3 — Tone by Channel
| Channel | Tone Setting | Notes |
|---|---|---|
| Website | Composed, editorial | Minimal punctuation excess |
| Email | Warm, direct | Slightly more personal than web |
| Instagram caption | Confident, brief | No hashtag overload |
| Instagram Stories | Casual but precise | Conversational, still on-brand |
| TikTok | Real, energetic | Can be lighter — still no slang |
| Product descriptions | Precise, desire-building | See `Product_Description_Voice.md` |
| Customer service | Warm, solution-first | Never defensive, never robotic |

## 06 — Digital Brand Rules

### 6.1 — Shopify / Website
| Rule | |
|---|---|
| Favicon | Brand mark only — not full wordmark |
| Page titles | Brand name at end: `Product Name — MvQueen` |
| Meta descriptions | Written in brand voice — never auto-generated |
| Button text | Direct action verbs: "Shop Now", "Get Yours", "View the Edit" — not "Click Here" |
| 404 page | Must be on-brand — not a default Shopify error page |

### 6.2 — Email
| Rule | |
|---|---|
| Subject lines | No all-caps, no excessive punctuation, no emoji overload |
| Preheader text | Always set — never leave as email preview default |
| Footer | Must include unsubscribe, address, and brand name |
| Image-to-text ratio | Minimum 60% text — for deliverability and accessibility |

### 6.3 — Social Media
| Rule | |
|---|---|
| Profile photos | Logo mark centered, no text |
| Bio | On-brand, current, no broken links |
| Hashtags | Max 5 branded + 5 niche — never 30-hashtag dumps |
| Tagging customers | Only with permission |
| Response time | Under 24 hours on DMs and comments |

## 07 — What Requires Doctrine Review

The following changes cannot be made without reviewing against `00_Doctrine/README.md` and the `Brand_Bible.md`:

- Introducing a new typeface
- Adding a color outside the approved palette
- Changing the logo or brand name spelling
- Launching a new sub-brand or product line
- Changing brand positioning language
- Updating the brand pillars

**Process:** Flag in `99_Command_Center/`, document the change, update affected files.

## 08 — Common Brand Violations & Corrections

| Violation | Correct Action |
|---|---|
| Canva template with wrong fonts | Replace with approved Google Fonts |
| Caption ending in "✨✨✨" | Trim to one emoji max or remove |
| Headline in all-caps | Title case for display, sentence case for body |
| Gold background on hero section | Switch to Vault Black or Clean Canvas |
| "Shop our amazing collection!" | Rewrite: "Shop the collection." |
| Product name in lowercase | Title case always |
| Blush text on white | Decorative color — change to Charcoal or Stone |
| Using Arial because font didn't load | Debug font load — do not accept fallback as final |

## 09 — Brand Rules for AI-Generated Content

When AI tools (Claude, ChatGPT, Jasper, etc.) produce content for MvQueen:

| Rule | |
|---|---|
| Always provide brand context | Link or paste relevant files before prompting |
| Always review before publishing | AI output is a draft, not a final |
| AI must not set tone policy | Humans approve all new brand language patterns |
| Contaminated language | Flag and rewrite any output that feels generic, hollow, or off-brand |
| Output must pass the pillar test | Does it reflect Quiet Confidence, Warm Luxury, or Feminine Precision? |
